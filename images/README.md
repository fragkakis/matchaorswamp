# Images Folder

This folder contains the images used in the Matcha or Swamp game.

## Structure

- `matcha/` - Contains images of matcha drinks
- `swamp/` - Contains images of swamps

## Finding Images

You can find suitable images from:
- **Matcha drinks**: Search for "matcha latte", "matcha drink", "green tea latte" on free stock photo sites
- **Swamps**: Search for "swamp water", "marsh", "bog", "murky water" on free stock photo sites

### Recommended Free Stock Photo Sites:
- Unsplash (https://unsplash.com)
- Pexels (https://pexels.com)
- Pixabay (https://pixabay.com)

## Image Format

- Recommended format: JPG or PNG
- The game will automatically resize images to fit
- For best results, use images that are at least 400x400 pixels

## Note

You can also use different image formats (png, jpeg, etc.) - just update the file extensions in `game.js` accordingly.
